AI Sentiment Analyzer (.NET 10)0.1 Project OverviewThis project is an automated AI tool designed to categorize customer reviews and feedback entries. It addresses the time-consuming process of manual categorization by providing instant sentiment feedback. The application classifies raw text into binary categories: Positive or Negative.0.2 System ArchitectureThe application follows a Three-Tier Architecture design:Presentation Layer: A Console-based User Interface for handling user input and displaying results.Business Logic Layer: C# backend code that processes strings and communicates with the prediction engine.Data/AI Layer: A trained ML.NET model containing the intelligence for sentiment classification.0.3 Tools & TechnologiesFramework: .NET 10.0.AI Library: Microsoft.ML (ML.NET Model Builder).Algorithm: FastTreeOva.IDE: Visual Studio 2026.0.4 Project FeaturesReal-time Prediction: Uses core integration logic to connect user input to the AI model for immediate results.Confidence Scoring: Displays the prediction result along with a confidence score to the user.Docker Integration: Includes a Dockerfile to ensure the app runs in any environment without dependency issues.0.5 Setup & InstallationRunning LocallyClone this repository to your local machine.Open the project in Visual Studio 2026.Build the project to restore dependencies.Run the application and enter a sentence when prompted (e.g., "This is a great app!").Running with DockerYou can use the provided Dockerfile to containerize the application:Bash# Build the Docker image
docker build -t ai-sentiment-analyzer .

# Run the application container
docker run -it ai-sentiment-analyzer
